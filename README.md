Лабораторная работа №1

Технологии и библиотеки
Язык программирования: C 
Компилятор: GCC 
Используемые библиотеки:
  <stdio.h> — стандартная библиотека ввода-вывода.

Содержание заданий (1-14)

1.Первая программа с переменными:О бъявление и инициализация переменных int, float, char с их последующим выводом.
2.Размеры типов: Определение объема памяти (в байтах), занимаемого типами char, int, float, double через оператор sizeof.
3.Переменные и память: Вывод значений переменных и их фактических адресов в оперативной памяти с использованием указателя &.
4.Остаток от деления: Реализация вычисления остатка от деления двух целых чисел с помощью оператора %.
5.Инкремент и декремент: Изучение работы префиксных и постфиксных операций (++a, a++, --a, a--).
6.Преобразование типов: Демонстрация целочисленного деления и явного приведения типов для получения дробного результата (на примере 5 / 2 = 2.5).
7.Потеря точности:Анализ специфики хранения вещественных чисел (0.1 + 0.2) с выводом 20 знаков после запятой.
8.Переполнение: Изучение поведения программы при выходе за границы диапазона типа int (прибавление 1 к 2147483647).
9.Вычисление формул: Реализация математических выражений: $x^2 + 2x + 1$ и $(x+1)^2 - (x-1)^2$.
10.Обмен переменных: Алгоритм смены значений между двумя переменными с использованием третьей (временной) переменной.
11.Последняя цифра числа: Использование арифметики по модулю 10 для извлечения последней цифры числа.
12.Разложение числа: Математическое выделение сотен, десятков и единиц из трехзначного числа.
13.Формат времени:Пересчет введенного количества секунд в формат часов, минут и секунд (HH:MM:SS).
14.Исследование памяти: Сравнение адресов нескольких переменных типа int для понимания расположения данных в стеке.

Инструкция по запуску

Для запуска программ необходимо наличие терминала и установленного компилятора gcc.

Компиляция
Для компиляции файла задания нужно ввести в терминале:
bash
gcc имя_файла.c -o программа

gcc task1.c -o task1

Запуск
После компиляции нужно выполнить команду:
bash
./название_программы
./task1
